# YuzuTask - 日系清新风 Chrome 新标签页插件

> 一款可爱灵动的 Chrome 新标签页插件，集成待办事项、番茄钟、工作记录和周报生成，帮你轻松管理日常工作。

---

## 产品定位

每次打开新标签页，看到的是你的专属工作台 —— 不需要额外打开任何 App，一个标签页搞定一天的任务管理、专注计时和工作记录，还能一键生成周报。

**视觉风格**：日系清新风 —— 柔和马卡龙色系（雾蓝、樱花粉、薄荷绿、奶油橘），简约线条图标，类似日系文具的温暖质感。

## 功能模块

### 1. 问候语 + 实时时钟

- 顶部显示根据时段变化的问候语和 emoji：
  - 清晨（5-9点）：🌅 早上好
  - 上午（9-12点）：☀️ 上午好，今天也要加油鸭~
  - 中午（12-14点）：🍱 中午好，记得吃饭哦~
  - 下午（14-18点）：💪 下午好，继续冲鸭~
  - 晚上（18-21点）：🌙 晚上好，辛苦一天啦~
  - 深夜（21-5点）：✨ 夜深了，早点休息吧~
- 右侧实时数字时钟，秒级刷新

### 2. 任务清单

**任务字段**：
- 标题、描述（可选）
- 优先级：高 / 中 / 低（分别用樱花粉 / 奶油橘 / 薄荷绿圆点标识）
- 分类标签：支持自定义，默认提供 `#工作 #开发 #会议 #文档 #沟通 #学习 #生活`
- 预计时长（分钟）
- 截止日期

**任务状态流转**：待办 → 进行中 → 已完成

**操作**：
- 添加任务（弹窗表单，支持 Enter 快速提交）
- 编辑任务（点击任务卡片打开编辑）
- 删除任务（hover 显示删除按钮）
- 标记完成（勾选圆形 checkbox，带弹性动画）
- 按状态筛选（全部 / 待办 / 进行中 / 已完成）
- 按标签筛选

**持久化**：所有数据通过 `chrome.storage.local` 保存在浏览器本地，不依赖任何服务器。

### 3. 番茄钟

**基础功能**：
- 默认 25 分钟专注 + 5 分钟短休息，每 4 个番茄后 15 分钟长休息
- 所有时长均可在设置中自定义
- 圆环进度条动画，颜色从雾蓝平滑过渡
- 开始 / 暂停 / 重置控制

**特色功能**：
- **任务联动**：开始专注前可选择关联一个待办任务，专注结束后自动将记录同步到"工作记录"
- **番茄计数**：已完成的番茄数以 🍅 图标展示
- **鼓励文案**：专注中随机显示日系鼓励小句子（如"专注是一种超能力~"、"慢慢来，比较快~"等）

### 4. 工作记录

**记录字段**：
- 工作内容描述
- 耗时（分钟）
- 日期
- 标签（复用自定义标签体系）

**数据来源**：
- 手动添加
- 番茄钟自动同步（选择关联任务完成专注后自动创建）

**查看方式**：
- 列表视图：按日期分组，每组显示当日总工时
- 日历视图：可切换周，有记录的日期显示小圆点标记
- 支持按日期筛选

**统计**：头部显示"今日总工时"

### 5. 周报生成器（亮点功能）

**核心价值**：老板每周五要收周报？YuzuTask 自动汇总你一周的工作记录和任务完成情况，生成结构清晰的周报，可编辑后一键复制。

**生成逻辑**：
1. 自动汇总本周（周一至周日）的所有工作记录
2. 按日期分组，每日列出工作明细和总工时
3. 汇总已完成任务和进行中任务
4. 按周报模板填充内容，输出为可编辑文本

**统计概览**：
- 本周工作记录数
- 本周已完成任务数
- 本周总工时

**周报模板**：
- 提供默认模板，支持占位符：
  - `{{completedItems}}` — 已完成事项
  - `{{inProgressItems}}` — 进行中的工作
  - `{{nextWeekPlan}}` — 下周计划
  - `{{blockers}}` — 需要协调的事项
- 用户可完全自定义模板内容

**提醒设置**：
- 可配置提醒生成周报的时间（如每周五 08:00）
- 到时间自动弹出 Chrome 通知

**操作流程**：点击"生成本周周报" → 自动汇总并填充 → 可直接编辑文本 → 点击"复制"到剪贴板

### 6. 通知提醒

通过 `chrome.alarms` + `chrome.notifications` API 实现：

- 番茄钟结束提醒："专注时间结束啦，休息一下吧~"
- 周报生成提醒："该写周报啦~ 点击新标签页生成本周周报吧！"

### 7. 设置页

- 番茄钟时长配置（专注 / 短休息 / 长休息 / 长休息间隔）
- 自定义标签管理（添加 / 删除）
- 数据管理（本地存储说明、清除数据）

## 技术栈

| 项目 | 选择 | 理由 |
|------|------|------|
| 框架 | React 18 + TypeScript | 生态成熟，类型安全 |
| 构建 | Vite | 快速构建，简洁配置 |
| 样式 | Tailwind CSS | 原子化 CSS，快速实现一致的配色 |
| 动画 | Framer Motion | 声明式动画，灵动的交互效果 |
| 图标 | Lucide React | 线条风格，契合日系简约感 |
| 日期 | date-fns | 轻量，支持中文 locale |
| 存储 | chrome.storage.local | 本地持久化，无需后端 |
| 插件规范 | Manifest V3 | Chrome 最新标准 |

## 数据模型

```typescript
// 任务
interface Task {
  id: string;
  title: string;
  description?: string;
  priority: 'high' | 'medium' | 'low';
  tags: string[];
  status: 'todo' | 'in_progress' | 'done';
  estimatedMinutes?: number;
  dueDate?: string;
  createdAt: string;
  completedAt?: string;
  order: number;
}

// 工作记录
interface WorkLog {
  id: string;
  content: string;
  tags: string[];
  durationMinutes: number;
  date: string;
  relatedTaskId?: string;
  createdAt: string;
}

// 番茄钟记录
interface PomodoroSession {
  id: string;
  taskId?: string;
  startedAt: string;
  endedAt: string;
  durationMinutes: number;
  type: 'focus' | 'break';
}

// 周报配置
interface WeeklyReportConfig {
  enabled: boolean;
  dayOfWeek: number;       // 0=周日, 5=周五
  time: string;            // "08:00"
  template: string;
  dateRange: 'mon-sun' | 'sun-sat';
}

// 用户设置
interface UserSettings {
  pomodoroFocusMinutes: number;
  pomodoroBreakMinutes: number;
  longBreakMinutes: number;
  longBreakInterval: number;
  weeklyReport: WeeklyReportConfig;
  customTags: string[];
}
```

## 安装使用

1. `npm install` 安装依赖
2. `npm run build` 构建生产包
3. 打开 Chrome → `chrome://extensions/` → 开启"开发者模式"
4. 点击"加载已解压的扩展程序" → 选择 `dist/` 目录
5. 打开新标签页即可使用
