## 商店信息

- **扩展名称**: YuzuTask - 任务事项 & 番茄钟 & 工作记录
- **简短描述** (132字符内):
  - 英文: Task manager, pomodoro timer, work log & weekly report — all in your new tab.
  - 中文: 任务清单、番茄钟、工作记录与周报生成，一个新标签页搞定一天的工作管理。

- **详细描述**:

### 英文版

```
YuzuTask replaces your new tab with a beautiful Japanese-style productivity workspace.

✨ FEATURES

📋 Task Management
• Create, edit, and organize tasks with priorities (high/medium/low)
• Custom tags for categorization (#work #dev #meeting #doc #study)
• Status tracking: todo → in progress → done
• Filter by status and tags
• Estimated time and due date support

🍅 Pomodoro Timer
• 25-min focus + 5-min break cycles
• Link pomodoro sessions to tasks for automatic work logging
• Circular progress animation
• Encouraging messages during focus time

📝 Work Log
• Manual or automatic logging (from pomodoro sessions)
• Calendar view with weekly navigation
• Daily time summary

📊 Weekly Report Generator
• Auto-summarize your week's work records
• Editable report with one-click copy
• Configurable report template
• Weekly reminder notification

🎨 Design
• Soft pastel color palette inspired by Japanese stationery
• Smooth animations and micro-interactions
• Clean, minimal interface

🔒 Privacy
• All data stored locally in your browser
• No server, no account, no tracking
• 100% offline capable

---

Permissions explained:
• Storage: Save your tasks, logs, and settings locally
• Alarms: Pomodoro timer and weekly report reminders
• Notifications: Timer completion and reminder alerts
```

### 中文版

```
YuzuTask 将你的新标签页变成一个日系清新的效率工作台。

✨ 功能特性

📋 任务管理
• 创建、编辑、整理任务，支持高/中/低优先级
• 自定义标签分类（#工作 #开发 #会议 #文档 #学习）
• 状态追踪：任务 → 进行中 → 已完成
• 按状态和标签筛选
• 支持预估时长和截止日期

🍅 番茄钟
• 25分钟专注 + 5分钟休息的经典循环
• 可关联任务，专注结束自动生成工作记录
• 圆环进度动画
• 专注时随机显示鼓励小句子

📝 工作记录
• 手动添加或番茄钟自动同步
• 日历视图，按周导航
• 每日工时统计

📊 周报生成器
• 自动汇总一周工作记录和任务完成情况
• 可编辑，一键复制到剪贴板
• 支持自定义周报模板
• 可配置每周提醒

🎨 设计风格
• 日系清新马卡龙配色，柔和温暖
• 丝滑的动画与微交互
• 简约优雅的界面

🔒 隐私安全
• 所有数据保存在浏览器本地
• 无需服务器、无需注册、零追踪
• 完全离线可用

---

权限说明：
• Storage：在本地保存你的任务、记录和设置
• Alarms：番茄钟计时和周报提醒
• Notifications：计时结束和提醒通知
```

## 截图说明

需要 1~5 张截图，建议尺寸 1280x800 PNG：

1. **首页** — 番茄钟 + 任务清单并排视图
2. **工作记录** — 列表视图 + 日历视图
3. **周报** — 周报生成与编辑界面
4. **设置** — 番茄钟配置 + 标签管理
5. **任务管理** — 添加任务弹窗 + 筛选功能

> 📸 截图需要你手动在新标签页中操作并截图，建议使用 Chrome DevTools 的设备模拟确保 1280x800 分辨率。
